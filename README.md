# 🎙️ Virtual Assistance Email for Blind People

A smart voice-enabled email assistant designed especially for visually impaired users.  
This project allows users to send, receive, and manage emails using voice commands instead of traditional typing and screen navigation.

---

## 📌 Project Overview

The **Virtual Assistance Email System for Blind People** helps visually challenged users interact with email services independently through speech recognition and text-to-speech technology.

The system converts:
- 🎤 Voice → Text (for composing emails)
- 🔊 Text → Voice (for reading emails)

This makes email communication accessible, simple, and user-friendly.

---

# ✨ Features

- 🎙️ Voice-based email composing
- 🔊 Read incoming emails aloud
- 📧 Send emails using speech commands
- 🗣️ Speech-to-Text conversion
- 🔈 Text-to-Speech conversion
- 🔐 Secure login authentication
- 📥 Inbox email reading
- 📤 Sent mail management
- 👨‍🦯 User-friendly interface for blind users
